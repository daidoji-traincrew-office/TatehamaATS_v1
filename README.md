# 館浜電鉄ダイヤ運転会用　車上装置
## キーボード操作一覧

### 主画面
AまたはL：ATS表示器切替  
R：列番設定器切替  
K：運転告知器切替  
D：ATS表示器と運転告知器切替  
  
### 列番設定器  
英数字：対応する英数字キー（テンキー対応）  
J：だんじり  
L：特  
-（マイナス）：回  
^（キャレット）：試  
¥（円記号またはバックスラッシュ）：臨  
@（アットマーク）：列番設定  
[（大括弧開始）：両数設定  
;（セミコロン）：時刻設定  
:（コロン）：運行設定  
]（大括弧終了）：停車設定  
O：停車  
P：通過  
V：Ver.表示  
Enter：設定/進  
BackSpaceまたはDelete：消去/戻  
Esc：取消
